# Symfony Alpine Docker
A Dockerfile with Alpine Linux that is ready for Symfony Application

Source: https://github.com/therceman/symfony-alpine-docker

## Dockerfile Bundle
- Alpine Linux 3.15
- Apache/2.4.52 with mod_php
- PHP 7.4.27 with OPcache
- Node.js 16.x LTS
- Composer (latest)
- Symfony CLI (latest)
